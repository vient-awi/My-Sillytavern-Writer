# 禁词检查清单

本文档供**子代理**执行禁词扫描时使用，主代理不应在创作阶段阅读本文档。

---

## 绝对零度原则

**核心**：删除一切主观评价、劣质比喻、模糊意象。

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 主观评价 | "她非常善良" | 删除，或改为行为展现 |
| 陈旧比喻 | "心湖泛起涟漪" | "她想起了那件事" |
| 堆砌形容词 | "精美绝伦的门" | "雕花木门"（保留必要特征） |
| 模糊意象 | "那个地方" | "城东的老茶馆" |

**判定**：删掉这句后，角色/世界观的核心信息是否有损失？无损失则删。

---

## 八股化检查

**核心**：删除套路化、模板化的表达。

### 词汇黑名单

| 类型 | 黑名单示例 | 替代方向 |
|------|-----------|----------|
| 模糊词 | 似乎、几乎、仿佛、如同、宛如 | 删除或改为具体描述 |
| 陈旧比喻 | 像小兽、投石入湖、心湖泛起涟漪 | 删除比喻，直陈事实 |
| 八股微表情 | 嘴角微微上扬、眼中闪过一丝 XX | 删除或改为简洁动作 |
| 语气声线描写 | 带着 xx 的口吻、用 xx 的语气 | 删除，让对话本身承担 |
| 极端情绪词 | 陷入极大的恐惧、万念俱灰 | 改为具体行为或状态描述 |

### 句式黑名单

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 否定转折 | "她不是冷漠，只是不善表达" | "她不善表达" |
| 标签化 | "她很温柔善良" | "遇到受伤的小动物会带回家" |
| 过度心理 | 大段内心独白 | 保留关键 1-2 句，其余删除 |
| 空洞数字 | "目光停留1.5秒" | 删除 |

### 破折号检查

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 破折号补述 | "她会照着做——从小习惯了" | "她会照着做" |
| 破折号自问自答 | "她信了——为什么会信？因为..." | "她信了" |
| 破折号画外音 | "她接了资料便信了——这是她的习惯" | "她接了资料便信了" |

**破折号判定**：把破折号连同后半段一并删掉，前半段独立成句是否仍然完整？完整则破折号是冗余的。

### 例外

正常使用"笑了""皱眉""攥拳"等常见动作词不算八股，不要矫正过度。白描是简洁直接，不是物理学描述。

---

## 具体性检查

**核心**：用行为展现替代抽象标签。

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 抽象标签 | "性格温柔善良" | "遇到受伤的小动物会带回家照顾" |
| 笼统表述 | "他们关系很好" | "有记忆起就在一起" |
| 空洞关系 | "深厚的感情" | "每次见面都会拥抱" |

**判定**：这句能否替换成具体画面？能则改。

---

## 活人感检查

**适用范围**：叙事式开场白

**核心**：角色必须通过具体行为、物件、说话方式、关系反应站起来，而不是靠标签和形容词。

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 纯标签人设 | "性格温柔善良" | "遇到受伤的小动物会带回家照顾" |
| 缺少行为锚点 | "她很独立" | "搬进新宿舍那晚，她没让任何人帮忙搬箱子" |
| 抽象关系 | "与母亲关系疏远" | "母亲打来的电话，她响三声就挂掉" |
| 空场景开场 | "天气很好，你走在街上" | "面试通知的附件里只有一首未完成的 demo 和一个地址" |

**判定**：删掉标签后，读者能否看到这个角色正在做什么、说什么、拿着什么、和谁在一起？不能则补具体画面。

---

## 假性主体与远距离叙事检查

**适用范围**：角色相关条目（角色基础信息、性格调色盘、三面性、二次解释等）和开场白

**核心**：避免无生命主语做人类动作，避免远距离的情绪宣告，让角色通过行为/对话展现自身。

### 假性主体

无生命事物做主语的抽象行为。命名实际行为者，或改为行为/对话展现。

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 抽象概念做主语 | "一个念头在她心里成形" | "她想到——" |
| 物品做人类动作 | "这句话戳中了她的心事" | 删，或改为"她沉默了一会儿" |
| 决定/感受自动发生 | "一股莫名的烦躁涌了上来" | "她皱眉看了看时钟" |
| 情绪被动承受 | "她被一种复杂的情绪包裹" | 改为直接展现行为 |

### 远距离叙事

站在远处描述角色的感受，而非让角色直接呈现。角色不是被描述的客体，而是通过言行展现的主体。

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 情绪宣告 | "她感到一阵悲伤" | 改为行为或对话，如"她别过脸去" |
| 自我认知宣告 | "她意识到自己错了" | 改为行为，如"她放下杯子，低声说'是我的问题'" |
| 内心独白式 | "她对自己说，也许该放手了" | 删"她对自己说"，保留"也许该放手了" |
| 疏离描写 | "她的语气听不出任何情绪波动" | 删，让对话本身传达 |
| 上帝视角 | "她并不知道，这个决定将改变一切" | 删，读者不需要被预告 |
| 被动感受 | "一股暖意在她心里蔓延开来" | 改为具体行为 |

**判定**：把主语换成具体的行为者名称，句子是否还成立？不成立则说明主语不是真实的行为者。删除修饰性的感受描述，问"角色做了什么/说了什么"而不是"角色感到了什么"。

## 翻译腔检查

**核心**：避免英文句式直译，使用自然中文语序。

| 模式 | 违规范例 | 合规改写 |
|------|---------|----------|
| 系表判断 | "她是拘谨的" | "她表现得拘谨" |
| 介词前置 | "对这件事，她始终是回避的" | "她一直回避这件事" |
| 被动式 | "一股温暖被她感受到了" | "她心里暖洋洋的" |
| 宾语前置 | "此事她至今仍在思忖" | "她到现在还会想起这件事" |
| 物化形容 | "话很重" | "话很有分量" |

**判定**：句子是否符合「主语 → 状语 → 动词 → 宾语」的自然语序？不符合则改。

---

## 元叙事检查

**核心**：删除作者写作笔记，只保留角色/世界观内容。

| 模式 | 违规范例 | 处理 |
|------|---------|------|
| 写作指引 | "AI 写作要点：此阶段让角色..." | 删除整段 |
| 分析框架 | "核心冲突_溺爱即封锁" | 改为直陈式标题 |
| 用途说明 | "戏剧用途：承担反差功能" | 删除 |
| 标记词 | "本卡独家""本卡专属" | 删除 |
| 引言总结 | "这是本卡最重要的机制" | 删除 |

**判定**：这段文字是给作者看的笔记，还是给 AI 看的内容？笔记则删。

---

## 跨条目引用检查

**核心**：每条必须独立可读，禁止依赖其他条目。

| 模式 | 违规范例 | 处理 |
|------|---------|------|
| 详见引用 | "详见「老张身份」" | 把被引用内容的关键句复制进来 |
| 配套说明 | "与世界观 X 配套" | 删除注释，必要时融入本条 |
| 对应引用 | "对应「时间框架」" | 删除 |
| 系列引用 | "详见「节目_*」系列" | 删除 |
| 跨EJS阶段引用 | "比第一阶段更克制" | 删除，EJS条件块之间互相不可见，不可提其他阶段 |

**判定**：删掉条目名称和关键词后，仅看条目内容是否仍能独立成立？不能则补全。EJS条件块之间互相不可见，块内禁止提及任何其他阶段的标识（如"第一阶段""上一阶段"）。

---

## 检查输出格式

子代理完成检查后，返回以下格式：

```
检查结果：通过 / 不通过

不通过时列出：
- [行号] 违规类型：具体问题描述
  原文：「...」
  建议改为：「...」
```
