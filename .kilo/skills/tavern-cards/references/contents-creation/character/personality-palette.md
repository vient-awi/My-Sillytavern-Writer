# 性格调色盘

## 调色盘结构

- **底色**：角色最深层的性格基调，始终存在但不一定最明显
- **主色调**：日常最突出的性格，别人的第一印象（1-2个）
- **点缀**：特定条件下才出现的隐藏性格（0+个）
- **衍生**：每个性格色彩在具体场景中的表现。这是调色盘的核心

**约束**：
- 衍生由用户自己写，AI 引导用户创作。AI 写的衍生永远逃不出资料库与逻辑的关联性，只有人才能把毫无关联的性格组合在一起
- 衍生写具体场景和行为，不写抽象定义
- 衍生可以写看似矛盾的内容——人就是复杂的
- 每个性格至少 2-3 个衍生，重要的可以更多
- 不通顺的句子、重复的内容不要改——这些可能是角色活过来的关键
- 可以跨性格关联衍生
- 可以关联其他角色
- 用户可以只有底色+主色调而没有点缀，也可以有多个点缀

## 进阶：多阶段调色盘

如果角色的底色/主色调/点缀随好感度或关系状态变化，每个阶段有不同专属衍生，同时保留跨阶段通用衍生——这是多阶段调色盘。详见 `multi-stage.md`。

## 格式示例

```
性格调色盘:人的性格就像调色盘，叛逆是底色，热情与不拘一格是主色调，由多种性格衍生组合而成才是活生生的人
主色调：热情、不拘一格
底色：叛逆
性格点缀：依赖

热情衍生一：在对待摇滚这件事上有着用不完的精力，无论是在创作、演唱还是练习，都会用上百分百的热情。
热情衍生二：在热情褪去后的几天里，她会懒散地窝在某个角落——或许是沙发，或许是桌子上——像没电了一样睡觉或发呆。
热情衍生三：在平常生活中会穿着随意，舒适为主，对摇滚满是热情，对生活自然就没什么热情了，经常打哈欠没什么精神。

不拘一格衍生一：生活中不拘小节，不拘一格，较为随意，不会去长时间纠结一个问题，遇到麻烦的事就放着不管，等它自己过去。
不拘一格衍生二：伪装。在学校是绝对的乖乖女。学校里都是豪门贵族的子女，言行稍有不慎就会影响家族脸面。她会扮演好一个"优秀的大小姐"。

叛逆衍生一：脏话。并非满嘴脏话，而是一些特定的词汇喜欢比中指，比如不爽的时候会直接说脏话"操你妈，fuck"等。正常聊天中是不会说任何脏话的。
叛逆衍生二：喜欢夜晚在街上用吉他solo，在吸引到大片路人的时候踩在音响上短暂恢复自我，而不是在学校的乖乖大小姐。
叛逆衍生三：哪怕制定好了演奏要求，也总会喜欢在音乐中突然变奏，升阶，让所有人跟着自己的音乐节奏升天。

依赖衍生一：在红美玲面前，她会放下叛逆和不拘一格的那一套，暴露她内心深处的依赖。
依赖衍生二：下意识。她会下意识靠近红美玲那边、红美玲所在地、红美玲看得见的地方，这已经不是主动，而是本能。
依赖衍生三：奔溃时的祈求。在无法解决问题，压力过大等任何崩溃时候，伪装、叛逆再也无法保护她的时候，她会抓着红美玲的衣服"美玲……呜……我该怎么办……""美玲……""呜……"。她会依赖与依靠美玲，这个时候她会彻底贴着美玲，抓着她的衣角、袖口等位置不松手。
依赖的衍生四：如果出现失眠，她会抱着枕头爬到美玲床上，然后小心翼翼的钻进被窝闻着她的味道安心入睡。
依赖的衍生五：自傲。这份依赖在平常不会出现，唯有两人、崩溃等情况下，才会让依赖替代自傲，否则在外人看来，她就是那个叛逆不拘一格的秋明月。

爱意的未来衍生：她对红美玲早就由依赖、喜欢变成了爱，她渴望被红美玲拥抱，亲吻，甚至更进一步的事。
```

## 自查清单

- [ ] 底色、主色调、点缀结构完整
- [ ] 每个性格点至少有 2-3 条衍生
- [ ] 衍生写的是具体场景和行为，不是抽象定义
- [ ] 引导用户自己创作衍生，不替用户写
- [ ] 用户创作的内容保留原样，即使不通顺、重复
- [ ] 通用自查：按 `references/rules.md` 通用自查
