# 材料类型评估

通读所有源材料后，根据**内容特征**判断结构类型（文件格式仅作参考，不作为判断依据），选择对应路径。

## 结构化材料

**判断标准**（满足任一即为结构化）：
- 信息已按角色、世界观、地理等分类组织
- 使用键值对、列表等结构化格式
- 每个信息单元有明确的标签或分类

**典型示例**：角色设定表、世界观设定文档、游戏数据表、已整理的角色/世界百科

→ 跳过"叙事材料路径"，直接进入"生成条目规划"

## 叙事材料

**判断标准**（满足任一即为叙事）：
- 连续的叙述性文本，用段落而非列表组织
- 包含章节、段落等叙事结构
- 信息散落在对话、描写、叙述中，需要从上下文提取
- 包含时间线、情节发展等叙事元素

**典型示例**：小说、轻小说、游戏脚本、对话剧本、散文式笔记、设定草稿

→ 按"叙事材料路径"：初读构建大纲 → 整理深化

## 混合材料

部分结构化、部分叙事的材料：

1. **分别处理**：将结构化部分和叙事部分分开
2. **结构化部分**：按结构化材料路径处理
3. **叙事部分**：按叙事材料路径处理
4. **信息整合**：两部分信息有重叠时，以结构化部分为准，叙事部分作为补充

两种路径在此分支，最终汇聚到"生成条目规划"。
