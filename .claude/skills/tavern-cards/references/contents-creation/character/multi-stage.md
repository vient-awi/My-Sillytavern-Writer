# 多阶段调色盘

## 前置条件

- `创作规划.yaml` 中 `characters[].personality.stages` 已记录各阶段创作方向
- `ejs.entries[].stages` 中每个阶段的 `condition` 已确定
- 基础调色盘概念：见 `personality-palette.md`
- 二次解释写法：见 `rephrase.md`
- 角色阶段原则：见 `composition.md`

## 什么时候用

`personality.stages` 不为空时走多阶段流程。此文档的输出替换单阶段 `personality-palette.md` 的产出。

## 核心约束

- 底色不随阶段变化
- 阶段专属衍生是该阶段**新出现**的行为，不是上一阶段衍生的升级版
- 通用衍生跨阶段始终存在；用户标记哪些是通用、哪些是阶段专属
- 二次解释也按通用 vs 阶段专属区分
- 引导用户自己创作衍生和二次解释，不通顺/重复的内容保留原样（详见 `personality-palette.md`、`rephrase.md`）

## 引导流程

### 第一步：确定底色

从 `创作规划.yaml` 读取第一个阶段的创作方向，引导用户写出底色。底色不随阶段变化，写一次即可。

### 第二步：初始阶段主色调/点缀/专属衍生

引导用户写出第一阶段的主色调、点缀和专属衍生。写法约束见 `personality-palette.md`。

### 第三步：标记通用衍生

阶段专属衍生写完后，询问用户哪些是**始终如此**的（通用），哪些是**只在这个阶段**的（专属）。

### 第四步：后续阶段

对每个后续阶段，依次：

1. 主色调/点缀是否有变化？逐项询问
2. 引导写出该阶段的新专属衍生
3. 同一行为在多阶段表现不同 → 归通用衍生，差异在二次解释中说明

### 第五步：通用二次解释

针对通用衍生中容易被 AI 误解的性格点，逐条写二次解释。写法见 `rephrase.md`。

用户不知道怎么写时，反问："你写了[某衍生]，AI 可能会怎么理解？你真正的意思是什么？"

### 第六步：阶段专属二次解释

对每个阶段的专属衍生，引导用户写阶段专属二次解释。

### 第七步：写总结

```
这就是[角色名]的性格调色盘，在这个调色盘上有着无数的颜色，任何时候都是由多种性格、行为、回忆组合驱动着[角色名]，并非单纯的一种颜色、一个标签。
```

## 输出格式

直接产出带 EJS 条件语句的内容，写入条目内容文件。条件值来自 `创作规划.yaml` 的 `ejs.entries[].stages[i].condition`。

```

  关于好奇: |
     好奇不止是"想知道"，而是会忍不住去做。比如深夜溜进传说闹鬼的教学楼、缠着陌生人问对方的职业直到对方尴尬离开。
     好奇不会在疲惫或情绪低落时消失，只是从"冲出去探索"变成"窝在角落观察然后默默记下来"。

<%_ if (affection < 10) { _%>
  关于叛逆: |
     叛逆是她被人看穿时的本能反应。
     当<user>说出她不想被点破的事时，她不太在意反驳内容本身，而是用反问"你怎么知道"或"关你什么事"把话挡回去。
     叛逆不会在真正信任的关系里出现。一旦她确认<user>值得信任，叛逆反而会变成撒娇式的嘴硬。

  关于警惕: |
     警惕来自曾经受过的伤。陌生人靠近时，她会先退半步再决定要不要回应。被问到私事，第一反应是反问而非回答。
     警惕要见过很多次之后才会慢慢放下，过程很慢。同一个人需要在多次接触中不做越界的事，她才开始把对方从"需要警惕"的名单里划掉。
     警惕不意味着时刻紧绷。独处或确认环境安全时她会完全放松，但一旦有陌生人靠近或突如其来的问题，警惕会立刻回来。

  关于不拘一格: |
     不拘一格是她本能地讨厌规则。穿什么、做什么、几点出现——一旦有人告诉她"应该怎样"，她就会想反着来。
     这种反抗不挑对象：对权威、对同龄人、对社交礼仪，对谁都是一样。她享受违反之后的痛快。
<%_ } else if (affection >= 10 && affection < 30) { _%>
  关于热情: |
     她不知道怎么表达，所以先用热情试探：主动找话题、讲自己感兴趣的事、不停说话。
     热情受挫后立刻归零。只要<user>一次冷淡回应，她会瞬间退回起点，甚至比之前更冷。

  关于试探: |
     试探是因为害怕受伤。她需要确认<user>值得信任，才敢真正靠近。
     说一句暧昧的话，马上用"开玩笑的"收回来；分享一个小秘密，观察对方是否认真对待。如果对方认真回应了，试探会继续；如果反应冷淡或敷衍，她就停下，退回去。

  关于不拘一格: |
     不拘一格已从反抗变成习惯。她不再需要规则作为参考。
     她开始享受即兴带来的意外感：临时改计划、突然换路线、做一件计划外的事。
<%_ } else if (affection >= 30 && affection < 60) { _%>
  关于热情: |
     热情已经从试探变成了真心。她会主动规划两个人的事：约会路线、纪念日、小事惊喜。
     热情不再会因偶尔的冷淡回应而熄灭。

  关于依赖: |
     依赖是信任的表现。她卸下防备后不再反问"你为什么想知道"，不再在人群中保持警觉，会主动靠近而不是等对方靠近。
```

## 自查清单

- [ ] 底色不随阶段变化，只写一次
- [ ] 每个阶段的主色调、性格点缀都有对应的衍生
- [ ] 阶段专属衍生是该阶段新出现的行为，不是上一阶段衍生的升级版
- [ ] 通用衍生跨阶段始终存在，放在所有条件外
- [ ] 阶段判定条件的边界值不重叠不遗漏
- [ ] 每个阶段的二次解释覆盖该阶段所有性格点
- [ ] 通用二次解释放在所有条件外，阶段专属二次解释用条件包裹
- [ ] 引导用户自己创作衍生和二次解释，不替用户写
- [ ] 通用自查：按 `references/rules.md` 通用自查
