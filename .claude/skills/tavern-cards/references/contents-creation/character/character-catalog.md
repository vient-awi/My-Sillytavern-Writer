# 角色速览

角色速览是角色索引，不是详细人设。只包含基本识别信息，方便 AI 快速定位"谁是谁"。

## 约束

只保留识别信息，不包含性格、外貌等详细内容。
必须包含姓名、性别、年龄。可选：身份、职务、班级、关系。
5 个以上角色时编写角色速览，5 个以下不需要。
排序——<user>（如有）最前，主要角色次之，NPC 最后。
5 个以下角色时，核心角色条目和 NPC 条目本身已足够辨识，不需要额外的索引条目。

## 格式示例

```
角色总览:
  - 姓名: '<user>'
    性别: 男
    年龄: 17
    班级: 高二3班

  - 姓名: 林小雨
    性别: 女
    年龄: 17
    班级: 高二3班
    备注: 与<user>同班

  - 姓名: 王老师
    性别: 女
    年龄: 35
    职务: 班主任
    负责班级: 高二3班

学校信息:
  学校名称: 市立第三高中
  年级设置: 高一、高二、高三
  学制: 三年制
```
