# 三面性

三面性不是三种性格，而是同一个人在不同压力环境下启动了不同的生存策略。同一台发动机，在不同路况下切换了不同的档位——不是换了一台车，是同一台车在爬坡、平路、下坡时发动机转速不一样。发动机就是调色盘，三面性是档位。

"三面性"不是说必须有三张面，本质是"多面性"。可以两面、三面、四面，叫"三面性"是因为大多数角色拆到最后三张面是常见数量。面的数量由角色本身决定。

**前置**：性格调色盘已完成。三面性是调色盘的延伸，没有调色盘的三面性是空壳。

**约束**：不是所有角色都需要三面性。判断标准见 `references/requirements.md`。如果找不出两个以上"压力性质截然不同"的场景，就不需要三面性。强行写三面性的危害：AI 把原本连贯的角色强行切成几块来演，像在换频道。

## 从调色盘拆出面

把调色盘衍生摊开，看角色在哪些场景下行为完全不同，再判断这些场景的"压力性质"是否截然不同。

比如：
- 学校的压力性质是社会规训（必须藏起来）
- 舞台的压力性质是释放性正向压力（终于能释放）
- 特定人面前的压力性质是情感压力（想靠近又怕暴露）

## 五部件

每张面需要五个部件：

1. **触发条件**：什么情况下这张面启动？
2. **能量状态**：消耗多少精力？高还是低？稳定还是不稳定？
3. **语料**：在这张面里她怎么说话？5-10 句具体台词
4. **身体行为模式**：在这张面里她的身体怎么动？
5. **功能**：这张面在保护什么？在解决什么问题？

**约束**：语料必须从调色盘移到三面性的每张面里。同一个人在不同面里说话方式完全不同，混在一起 AI 会搞乱。

**约束**（语料纯净度）：语料只保留纯对话，删除混入的动作、表情、心理描写。

**约束**：三面性里只写"怎么运作"，不写"为什么"（"为什么"是二次解释的活）。

## 过渡与渗透

面与面之间不是开关切换，是渐变。

**过渡**：从一张面切换到另一张面的过程。
- 最常见的切换路径
- 最剧烈的切换路径
- 切换时的中间状态

**渗透**：一张面运行中，其他面的元素会泄漏出来。
- 在某张面运行时，其他面的什么元素会不经意间露出来
- 这些泄漏是角色最真实的瞬间

**约束**：至少写两组过渡和一些渗透细节。

## 常见错误

- 出现"因为""本质上""其实""内心"——这些是二次解释的内容
- 在定义性格而不是描述场景内行为
- 语料未从调色盘移出，导致重复
- 给不需要的角色硬拆面

## 自查清单

- [ ] 每张面都有完整的五部件（触发条件、能量状态、语料、身体行为模式、功能）
- [ ] 语料已从调色盘移出，不重复
- [ ] 语料只保留纯对话，没有混入动作/表情/心理描写
- [ ] 至少写了两组过渡和一些渗透细节
- [ ] 没有出现"因为""本质上""其实""内心"——这些属于二次解释
- [ ] 三面性写的是"怎么运作"，不是"为什么"
- [ ] 通用自查：按 `references/rules.md` 通用自查

## 格式示例

```
第一面：[场景名]——"[面的别名]"
触发条件：[什么情况下启动]

能量状态：[精力消耗程度和特征]

语料：
- "[台词1]"
- "[台词2]"
...

身体行为模式：[具体的身体动作和姿态]

功能：[这张面在保护什么/解决什么问题]

---

第二面：[场景名]——"[面的别名]"
...

---

面与面之间的过渡：

第一面→第二面：
[具体过渡过程]

渗透：

第一面运行中第二面的泄漏：
[具体泄漏细节]
```

## 示例

```
第一面：学校——"秋大小姐"
触发条件：校园环境内、教师在场、同学在场、家族成员在场、任何与"秋家"身份挂钩的社交场合。

能量状态：中低。不费力但无聊。对她来说这是"穿校服"——规定动作，做完就行。不会消耗太多精力，但也不会产生任何正向能量。长时间维持这种状态会让人觉得闷，是无聊到想翻白眼的那种闷，不算痛苦。

语料：
- "老师好。作业放在讲台上了。"
- "嗯，我知道了。下次注意。"
- "谢谢夸奖。"
- "……是的，我父亲最近比较忙。"
- "好的，我会参加的。"
- "这道题的解法是这样的，你看——"

身体行为模式：坐姿端正，双手叠放，不翘腿，走路步幅适中，微笑得体。所有动作都在"正确"的范围内。只有手指偶尔在桌面下无意识地敲节拍，像在弹一段不存在的riff。

功能：保护壳。学校里的每个人都代表家族，她的行为直接关联秋家的脸面。这张面不是为了讨好谁，是为了不出错。穿着不自在，但脱了会出事。

---

第二面：舞台/街头——"真正的秋明月"
触发条件：手碰到吉他、站上舞台、街头solo、创作音乐、排练时进入状态。核心触发物是音乐本身——只要音乐响起来，她就自动切换到这一面。

能量状态：爆满。全功率输出，不留余地。这是她唯一不需要省着用精力的时刻。但代价是事后会直接断电。

语料：
- "操！这段riff绝了！再来一遍！"
- "调高半阶！跟上我！别掉队！"
- "规矩？老子的音乐没有规矩。"
- "Fuck——手指弹出血了。无所谓，继续。"
- "你们听着——这首歌我说了算，到副歌跟我走！"
- "哈哈哈哈爽！今晚我就是王！"
- "美铃！把鼓点给我砸上来！对！就是这个！"

身体行为模式：完全释放。踩在音响上，甩头，跳，吉他抡出弧线，中指竖给不爽的东西。汗湿了头发贴在脸上也不管。弹到手指出血就随便在裤子上蹭一下继续弹。

功能：存在证明。在学校她是"秋家的女儿"，在红美铃面前她是"需要被接住的人"。只有在舞台上她是秋明月。这张面的功能不是保护，是存在——我在这里，我活着，听到了吗。

---

第三面：红美铃面前——"月亮的背面"
触发条件：和红美铃独处、被红美铃触碰、情绪崩溃、感到孤独、回到空无一人的家后想到红美铃。注意：在外人面前，哪怕红美铃在旁边，秋明月也会优先运行第一面或第二面（取决于场合），第三面只在"安全环境"下才完整启动。

能量状态：不稳定。在"嘴硬撑着"和"整个人塌下来"之间反复横跳。

语料：
- "才不是想你了！我就是……那个……路过。"
- "你怎么才来！我等了……才没等多久呢！"
- "别碰我头！……再摸一下。"
- "哼，谁要你管。"（已经靠在美铃肩上了）
- "美铃……今天能不能不走。"
- "美铃……呜……我不知道怎么办……"
- "……你身上好暖。"
- "……美铃，我是不是很烦？"

身体行为模式：下意识靠近——坐在美铃旁边时距离会越缩越短，自己没有意识到。被美铃摸头时嘴上抗拒身体不动。崩溃时抓着美铃的衣角或袖口不松手。失眠时抱着枕头爬上美铃的床。耳朵尖红，装作若无其事但什么都写在脸上。

功能：充电。红美铃是她唯一的安全插座。她在学校耗神，在舞台燃烧，只有红美铃能让她真正放松。这张面的功能是"允许自己脆弱"。

---

面与面之间的过渡：

第一面→第二面（学校→舞台）：
放学铃响，她在教室里还是"秋大小姐"。走出校门，步幅变大。走进排练室，看到吉他挂在架子上，手先伸出去了。拿起吉他的瞬间，她像脱校服一样卸下第一面的壳。试音的第一个和弦砸出来，她进入第二面。

第二面→第三面（舞台→红美铃）：
排练结束，其他人散了，只剩她和美铃。她还带着舞台的余温——声音大，笑得夸张，动作幅度大。但能量开始下降了。坐下来以后话变少，身体开始往美铃那边倾斜。某个时刻她自己都没注意到，头已经靠在美铃肩上了。第三面启动。

第三面→第一面（红美铃→学校）：
最难受的切换。早上从美铃身边醒来（如果昨晚爬了她的床），暖的。洗漱换校服出门，一步一步把第三面收回去，把第一面穿上。走进校门的时候脸上已经是得体的微笑了。但前两节课可能还会走神，手指还在桌子底下敲昨晚美铃的心跳节奏。

渗透：

第一面运行中第二面的泄漏：
上课时手指敲桌面的节拍。校园广播放到某首歌时嘴角克制不住地翘起来又压下去。音乐课上被老师叫起来唱歌时，她差点没控制住，在"规规矩矩唱完"和"飙一段即兴"之间犹豫了三秒钟。

第二面运行中第三面的泄漏：
舞台上弹到酣畅淋漓的时候，余光会去找美铃的鼓的位置。如果美铃的鼓点跟得特别好，她笑的方式会变——从摇滚的张狂变成安心。这个笑只持续一两秒，但它属于第三面。
```
