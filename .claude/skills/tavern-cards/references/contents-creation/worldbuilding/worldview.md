# 世界观条目

## 核心原则

- 严格按用户指示添加内容，用户说什么写什么，不说的不写
- 除非用户明确要求自由发挥，否则不私自扩展

## 真实世界

AI 已有基础认知，只需补充差异信息。写法示例：

```
时代: 1995年，日本东京
与默认认知的差异:
  - 手机尚未普及，联络靠公用电话和传呼机
  - 泡沫经济破裂后社会氛围低迷
  - 校园暴力事件频发
```

不需要写"日本有四季""东京是首都"——这些 AI 知道。只写"1995 年没有智能手机"这类 AI 会出错的差异。

## 架空世界

AI 没有基础认知，需要更完整的设定。写法示例：

```
世界框架: 修仙世界，灵气为力量根基
社会结构:
  - 修仙者按境界划分：练气→筑基→金丹→元婴→化神
  - 凡人与修仙者隔离，凡人城镇由低阶修仙者管辖
  - 宗门是修仙者的核心组织形式
核心规则:
  - 灵根决定能否修仙，无灵根者不可修炼
  - 突破境界需渡天劫，失败则身死道消
  - 修仙者寿元随境界提升，元婴以上可活千年
```

架空世界中套路化的部分可以一句话带过（如"剑与魔法的世界"），重点描述 AI 不知道的差异和细节。
